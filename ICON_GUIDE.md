## üìê Required Icon Sizes

### iOS App Store

#### App Store Icon (Required)
- **1024 x 1024 px** - PNG, no transparency, no rounded corners
- Color space: sRGB or P3
- This is the master icon shown in App Store listings

#### App Icon Sizes (Auto-generated by Xcode or manual)
If using Xcode Asset Catalog (recommended):
- **1024 x 1024** - Single image, iOS generates all sizes automatically

If generating manually:
- **180 x 180** - iPhone @3x (iPhone 14 Pro, 13 Pro, 12 Pro, X, XS, 11 Pro)
- **167 x 167** - iPad Pro @2x
- **152 x 152** - iPad @2x (iPad, iPad mini)
- **120 x 120** - iPhone @2x (iPhone SE, 7, 8, X, XS)
- **87 x 87** - iPhone @3x (Settings)
- **80 x 80** - iPad @2x (Spotlight, Settings)
- **76 x 76** - iPad @1x
- **60 x 60** - iPhone @2x (Spotlight)
- **58 x 58** - iPhone @2x (Settings)
- **40 x 40** - iPhone @2x, iPad @2x (Spotlight, Settings)
- **29 x 29** - iPhone @1x, iPad @1x (Settings)
- **20 x 20** - iPhone @1x, iPad @1x (Notifications)

**Note:** iOS automatically rounds icon corners. Don't add rounded corners yourself.

---

### Android (Google Play)

#### Play Store Icon (Required)
- **512 x 512 px** - 32-bit PNG with alpha (transparency)
- Full bleed (no padding/border required)
- Can have transparency but not recommended for app icons

#### Adaptive Icon (Required for Android 8.0+)
**Foreground:**
- **432 x 432 px** - 32-bit PNG with transparency
- Safe zone: Center 264 x 264 px (critical content must fit here)
- Full size: 432 x 432 px (can extend to edges for visual interest)

**Background:**
- **432 x 432 px** - PNG or solid color
- Can be transparent, solid color, or gradient
- Will be visible around foreground on different device launchers

**Current Setup in app.json:**
```json
"adaptiveIcon": {
  "foregroundImage": "./assets/adaptive-icon.png",
  "backgroundColor": "#00D084"
}
```

#### Launcher Icon Sizes (Auto-generated or manual)
- **192 x 192** - XXXHDPI
- **144 x 144** - XXHDPI
- **96 x 96** - XHDPI
- **72 x 72** - HDPI
- **48 x 48** - MDPI

**Note:** Android launchers apply various shapes (circle, square, rounded square, squircle). Design with safe zone in mind.

### Method 4: Expo Asset Generator (Easiest for Expo Projects)

Since you're using Expo, this is the **RECOMMENDED** method:

**Steps:**

1. **Update app.json with master icon:**
```json
{
  "expo": {
    "icon": "./assets/icon.png"  // 1024 x 1024 PNG
  }
}
```

2. **Create adaptive icon for Android:**
```json
{
  "expo": {
    "android": {
      "adaptiveIcon": {
        "foregroundImage": "./assets/adaptive-icon.png",  // 432 x 432 PNG
        "backgroundColor": "#00D084"
      }
    }
  }
}
```

3. **Build with EAS (generates all sizes automatically):**
```bash
# Install EAS CLI
npm install -g eas-cli

# Login to Expo account
eas login

# Configure project
eas build:configure

# Build for iOS (generates all icon sizes)
eas build --platform ios

# Build for Android (generates adaptive icons)
eas build --platform android
```

**Expo automatically generates:**
- All iOS icon sizes from `assets/icon.png`
- Android adaptive icons from `assets/adaptive-icon.png`
- No manual resizing needed!

## üé® Creating the Master Icon (1024 x 1024)

### Design Guidelines

**Safe Zones:**
- **iOS:** 1024 x 1024 (iOS adds rounded corners automatically)
- **Android Adaptive:** Keep critical content in center 264 x 264 px
- **Padding:** Leave ~10% padding from edges (prevent clipping)

**Color Guidelines:**
- **Primary:** #00D084 (Herb brand green)
- **Secondary:** White, light green gradients
- **Avoid:** Too many colors (3 max for clean look)
- **Contrast:** Ensure raccoon stands out from background

**Style Guidelines:**
- **Simple:** Should be recognizable at 29 x 29 px
- **Centered:** Keep focus centered for different launcher shapes
- **No Text:** Avoid text (illegible at small sizes)
- **Unique:** Stand out from other health/wellness apps

### Design Templates

#### Template 1: Circular Badge
```
- 1024 x 1024 canvas
- Circular green (#00D084) background (920 x 920 px, centered)
- Raccoon head centered (600 x 600 px)
- White outline around circle (20 px stroke)
- Subtle shadow for depth
```

#### Template 2: Gradient Background
```
- 1024 x 1024 canvas
- Radial gradient: #00D084 (center) ‚Üí #00A06A (edges)
- Raccoon full body (slightly smaller, 550 x 550 px)
- Centered on canvas
- Soft glow around raccoon
```

#### Template 3: Leaf Motif
```
- 1024 x 1024 canvas
- Solid green (#00D084) background
- Subtle cannabis leaf watermark in background (large, 70% opacity)
- Raccoon head in foreground (650 x 650 px)
- Overlapping composition
```

## üñºÔ∏è Creating Adaptive Icon for Android

### Foreground Image (432 x 432 px)

**Guidelines:**
- **Safe Zone:** 264 x 264 px (center) - critical content MUST fit here
- **Full Canvas:** 432 x 432 px - can extend for visual interest
- **Transparency:** Use alpha channel for non-rectangular shapes
- **Padding:** ~84 px from edge to safe zone

**Recommended Design:**
```
- 432 x 432 canvas with transparency
- Raccoon head centered at 264 x 264 px (safe zone)
- Can extend ears/details slightly beyond safe zone
- Export as 32-bit PNG with transparency
```

**Current File:** `assets/adaptive-icon.png`

### Background Color/Image (432 x 432 px)

**Option 1: Solid Color (Recommended)**
```json
"backgroundColor": "#00D084"
```
- Clean, simple
- No additional image needed
- Works well with all launcher shapes

**Option 2: Background Image**
```json
"backgroundImage": "./assets/adaptive-icon-bg.png"
```
- 432 x 432 PNG
- Can be gradient, pattern, or texture
- Must work with foreground at any shape
- Not recommended unless you have specific design needs
